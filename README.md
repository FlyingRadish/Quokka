# Quokka
A toolkit for Android developing

## Dependency
1. Add jitpack maven repo
```
repositories {
  ...
  maven { url 'https://jitpack.io' }
}
```
2. Add dependency
```
dependencies {
  compile 'com.github.houxg:Quokka:0.1'
}
```

## Usage
### CommandReceiver
Send custom command to your app via adb
### Print
Print every int enum
